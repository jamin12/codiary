# codiary
# 프로젝트 목적
- 마크다운 파일을 통해 깔끔히 정리된 페이지를 보여주며, 공부한 내용들을 저장 할 수 있는 공대생들을 위한 감각적인 블로그 제작

# 기술 스택 및 구조
- ![](./img/codiary-stack.png)

# 개발 환경
- 언어: javaScraipt
- 서버: nginX
- 프레임 워크: Express, node, react, sequelize
- 데이터베이스: mySql
- IDE: VS Code, dbeaver, postman
# 역할
## 강경민
- api 구현
- DB 설계
  - 트리거, 스케쥴러 활용
- 구글 로그인
- Cors문제 해결
- 입력값 검증 구현
- 유저 검증 구현
- 이미지 서버 구현
- 에러 핸들러 구현
- Redux활용해 프론트 로그인 기능 구현
- Axios로 백 프론트 연결
## 임효현
- UI/UX 디자인
- 웹 퍼블리싱
- Front - end
